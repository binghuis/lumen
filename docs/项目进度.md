## 一句话现状

**技术可行性已端到端验通**:真实 B站 直播间里跑通 `弹幕 → 豆包(人格+情绪)→ 火山语音 → Live2D 口型 → 导演层动作` 整条链路,含最难的"情绪→动作"导演层。距正式上线还差:首响延迟、正式推流、内容安全、好模型。

## 结论(2026-06-20 · 技术可行性验收)

- **技术命题(纯 AI 能不能搭出会聊、会说、会动、有反应的 B站 虚拟主播):成功 ✅。** 全链路在真实直播间端到端跑通,核心增量(导演层:情绪→动作)实测触发成功。
- **市场命题(纯 AI 陪聊在 B站 有没有人看、会不会打赏):待验。** 仅见零星真实信号(关注/点赞/几条弹幕),量太小,需持续真播 + 引流才能下结论。
- **一句话**:难的从来不是技术栈,栈已证明能通;接下来要赌的是内容 / 人格 / 市场。

## 已完成(均用真实服务验证过)

- **接收**:B站 直播间弹幕/礼物/醒目留言/上舰,实时(WebSocket 推送)。
- **大脑**:火山豆包 Ark,人格「流明」,会点名、接梗、按礼物/SC/上舰分别答谢,带短期记忆不复读。
- **声音**:火山 SeedTTS 2.0(可爱女声),**流式边出边播(ffplay)**;edge-tts 免费兜底。
- **闭环**:开了真直播间,观众发"你好/你是谁/讲个笑话"等弹幕,AI 都用语音即兴回应,人设在线。
- **音频上流(本机 Mac)**:BlackHole + 多输出设备,语音同时进扬声器和采集端。
- **形象(Live2D)**:VTube Studio 加载模型,BlackHole 音频驱动口型(真机验过会动)。
- **导演层(核心增量)**:豆包每条回复带 [情绪] 标签 → 解析剥离 → 触发 VTS 同名热键播放表情/动作。真机测:豆包出 [开心] → 模型做动作,实测成功。
- **端到端**:真直播间发弹幕 → AI 带情绪语音回应 + 口型 + 动作,一气呵成。

## 当前主要问题 + 解决思路

### 问题 1:正式推流上播堵死(Mac + 低粉)
- **现状**:OBS/ffmpeg/云推流都需要**推流码**,而推流码统一要 **≥5000 粉**;唯一不需推流码、无粉丝门槛的 PC 推流是 **B站直播姬,但只有 Windows 版**,Mac 装不了。手机扫码开播无门槛,但 AI 语音/画面合成进不去。
- **解决思路**:① 上一台 **Windows 主机(云 VM 或实体)跑直播姬 + 全栈**(代码可移植,改 `afplay`→Windows 播放、BlackHole→VB-CABLE)——推荐;② 涨到 5000 粉后 Mac 用 OBS;③ MVP 阶段先不上播,自己房间验内容。
- **优先级**:高(上线前必解)。

### 问题 2:首响慢 —— 已定位是豆包 TTFT,不是管线
- **流式已实现**:豆包 `stream=True` 按句吐 + 火山 SeedTTS 边出边播(ffplay),首句先开口。
- **实测定位**(排除代理/输入/代码):火山 TTS 首个音频分片 **0.57s**(快);**豆包首 token(TTFT)= 3~8s**,且整段≈首token。裸连 aiohttp(不走代理)、最小 prompt「你好」也一样 → **是豆包「按Token付费(免费额度)接入点」的服务侧延迟**,与网络/代码无关。
- **结论**:流式让 TTS 段更快、回复更顺,但**首响被豆包 TTFT 卡死,流式解不了**。
- **解法**:① 火山 **TPM保障包**(同云升档,花钱,确定有效);② 换云试别家免费档(DeepSeek/Qwen/MiniMax,不保证)。"免费又快"靠 A/B 碰运气,"保证快"得花钱。
- **优先级**:高。

### 问题 3:画面/动作"慢且不够灵活"(已做出来,待精修)
- **现状**:Live2D(VTube Studio)+ 口型 + 导演层动作**已跑通**。但体感慢/不利索,是三层叠加:① 首响被豆包 TTFT 拖(见问题2,**最大头**);② 口型是音量驱动(非音素);③ 动作有热键渐入时间 + Hiyori 自带动画偏慢 + 免费模型 rig 有限。
- **解决思路**:首响 → 问题2;口型 → 调平滑度 / 上音素口型;动作 → 调渐入时间、挑快动画、换"绑了手臂"的好模型。理论上限是 Neuro 级(实时灵活),可达,但分块攻克。
- **优先级**:中(精修项,不卡"技术可行性已验通"的结论)。

### 问题 4:内容安全 + AI 标注未做
- **现状**:弹幕原文直进 LLM、回复直接播,零防护。
- **解决思路**:入(弹幕)+出(回复)双向敏感词过滤;画面挂"AI生成"标注、开播前 10 秒语音+字幕表明身份。
- **优先级**:高(合规,上线前必须)。
